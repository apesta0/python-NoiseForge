# Purpose

NoiseForge is an open-source procedural generation project focused on:

- creativity
- experimentation
- learning
- collaboration
- accessibility

This Code of Conduct exists to help maintain a respectful, constructive, and welcoming environment for everyone participating in the project.

---

# Our Standards

Examples of positive behavior include:

- being respectful toward others
- accepting constructive criticism
- helping new contributors
- discussing ideas professionally
- encouraging experimentation and learning
- reporting bugs responsibly
- respecting differing technical opinions
- contributing useful documentation or fixes

---

# Unacceptable Behavior

The following behaviors are unacceptable within the NoiseForge community:

- harassment
- hate speech
- discrimination
- personal attacks
- trolling
- intimidation
- doxxing
- malicious code contributions
- intentionally disruptive behavior
- spam or self-promotion abuse
- insulting contributors or maintainers

This applies to:

- issues
- pull requests
- discussions
- commits
- external community spaces related to the project

---

# Contribution Expectations

Contributors are expected to:

- write understandable code
- document major changes
- avoid intentionally harmful code
- test contributions when possible
- keep pull requests focused and clean
- respect project structure and standards

---

# Beginner-Friendly Environment

NoiseForge welcomes:

- beginners
- students
- experimental developers
- self-taught programmers

Questions and learning-oriented discussions are encouraged.

---

# Constructive Criticism

Feedback should focus on:

- code quality
- maintainability
- architecture
- performance
- security
- usability

Avoid:

- insults
- hostility
- dismissive behavior
- gatekeeping

---

# Enforcement Responsibilities

Project maintainers may:

- remove harmful content
- reject malicious contributions
- restrict participation
- lock discussions
- ban repeat offenders

when behavior violates this Code of Conduct.

---

# Reporting Issues

If you encounter unacceptable behavior, report it to the project maintainer.

Include:

- screenshots if applicable
- reproduction details
- relevant links
- timestamps if possible

---

# Scope

This Code of Conduct applies to:

- GitHub repositories
- discussions
- issue trackers
- pull requests
- community spaces directly related to NoiseForge

---

# Open Source Respect

NoiseForge is an independent open-source project.

Respect:

- contributor time
- volunteer work
- experimentation
- creative direction

Constructive disagreement is welcome.

Hostility is not.
